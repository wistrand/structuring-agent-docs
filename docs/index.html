structuring-agent-docs

Structure a repository's documentation so AI coding agents navigate it efficiently.

A repeatable model: a human-facing README.md, an agent-facing CLAUDE.md entry point, and topic-focused agent_docs/ deep dives loaded on demand. Progressive disclosure applied to a whole repo.

TierFile(s)AudienceRole
User guideREADME.mdHumansWhat, why, quick start.
Entry pointCLAUDE.mdAgentsRead first: layout, commands, conventions, link index.
Deep divesagent_docs/*.mdAgentsOne topic per file, loaded on demand.
Download the skill (.zip) Read the skill Browse the repo

Install

Download and unzip into a Claude Code skills directory:

curl -L -o structuring-agent-docs.zip \
  https://wistrand.github.io/skill-agent-docs/structuring-agent-docs.zip
unzip structuring-agent-docs.zip -d ~/.claude/skills/

Or clone and symlink:

git clone https://github.com/wistrand/skill-agent-docs
ln -s "$PWD/skill-agent-docs" ~/.claude/skills/structuring-agent-docs

The agent loads it when you ask how to set up or reorganize CLAUDE.md, AGENTS.md, or agent_docs/ for a repository.

MIT licensed. The download contains the skill payload only (SKILL.md, references/, templates/), built from the repo by GitHub Actions.
